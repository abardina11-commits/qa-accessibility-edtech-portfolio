## What Distinguishes My Approach

A background in neuropsychology helps identify:
- cognitive overload in interfaces
- navigation issues
- accessibility barriers
- usability problems

This allows me to evaluate a product from the perspectives of quality, convenience, and user behavior simultaneously.

---

## Prioritization in My Work

Focus:
1. **Critical user scenarios** (login, registration, payment).
2. **Accessibility barriers** (WCAG 2.1 AA — showstopper violations).
3. **Cognitive risks** (navigation confusion, interface overload).
4. **Business impact** (what breaks conversion or causes user churn).

Each bug report includes:
- Risk
- User Impact
- Business Impact
- Why This Matters
