
# Чек-лист доступности (WCAG 2.1)

## 1. Клавиатурная навигация
- [ ] Все элементы доступны через клавишу Tab
- [ ] Видимый индикатор фокуса на активных элементах
- [ ] Нет ловушек фокуса (невозможно выйти с клавиатуры)

## 2. Контрастность текста
- [ ] Обычный текст: контрастность ≥ 4.5:1
- [ ] Крупный текст (18pt+): ≥ 3:1

## 3. Альтернативный текст
- [ ] Все изображения имеют атрибут alt
- [ ] Декоративные изображения: alt=""
- [ ] Сложная графика: подробное описание

## 4. Семантика и структура
- [ ] Правильная иерархия заголовков (h1 → h2 → h3)
- [ ] Формы имеют связанные label
- [ ] Таблицы имеют заголовки

## 5. ARIA-атрибуты
- [ ] Интерактивные элементы имеют role
- [ ] Динамический контент обновляется через aria-live

*Использовать для каждого тестирования доступности*
