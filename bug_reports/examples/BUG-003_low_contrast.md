# Bug ID: BUG-003

## Заголовок
Недостаточная контрастность текста на кнопке «Регистрация»

## Окружение
- Браузер: Chrome 123
- ОС: Windows 10
- Инструмент: WAVE, WebAIM Contrast Checker
- URL: https://example-edtech.com/register

## Приоритет
Medium

## Серьёзность
Medium

## Стандарт WCAG
1.4.3 Contrast (Minimum) — Level AA

## Шаги для воспроизведения
1. Открыть страницу регистрации
2. Проверить кнопку «Зарегистрироваться»

## Фактический результат
- Контрастность текста кнопки: 2.8:1
- Цвет текста: серый (#888888) на светло-сером фоне (#F0F0F0)

## Ожидаемый результат
- Контрастность ≥ 4.5:1 для обычного текста
- Цвет текста: тёмно-синий (#004080), фон: светло-синий (#E6F0FA)

## Частота воспроизведения
100%

## Влияние на пользователя
- Пользователи с нарушениями зрения не видят кнопку
- Риск потери конверсии при регистрации

## Рекомендации
- Изменить цвет текста на #004080
- Увеличить фон кнопки до контрастности ≥ 4.5:1

## Тип дефекта
Accessibility / Visual

## Связь с задачей Jira
EAQ-19 (проверка контрастности)
