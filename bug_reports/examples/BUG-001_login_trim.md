# Bug ID: BUG-EAQ-001

## Заголовок
Ошибка авторизации при вводе валидного email с пробелом в конце

## Окружение
- Браузер: Chrome 123
- ОС: Windows 10
- Среда: Demo / Staging
- URL: https://example-edtech.com/login

## Приоритет
High

## Серьёзность
Major

## Предусловия
- Пользователь зарегистрирован в системе
- Открыта страница авторизации

## Шаги для воспроизведения
1. Ввести email: test_user@example.com (с пробелом в конце)
2. Ввести валидный пароль
3. Нажать кнопку «Войти»

## Фактический результат
- Отображается ошибка: «Неверный email или пароль»
- Авторизация не происходит

## Ожидаемый результат
- Пробелы в начале/конце строки автоматически обрезаются (trim)
- Пользователь успешно авторизуется

## Частота воспроизведения
100%

## Влияние на пользователя
- Пользователь не может войти в систему из-за незначительной ошибки ввода
- Повышенная фрустрация, возможен уход с платформы

## Технические детали
Вероятно, отсутствует `trim()` для поля email перед отправкой запроса.

## Рекомендации
- Обрезать пробелы на клиенте (JavaScript) и/или сервере
- Добавить подсказку пользователю при ошибке

Functional / Usability
