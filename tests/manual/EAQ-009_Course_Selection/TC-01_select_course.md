# Test Case ID: TC-EAQ-003-01

## Название
Выбор курса из каталога и переход на страницу курса

## Модуль
Каталог курсов (Course Selection)

## Приоритет
High

## Предусловия
- Пользователь не авторизован (или авторизован)
- Открыта главная страница каталога: https://demoqa.com/books

## Шаги
1. Нажать на карточку первого курса в списке
2. Дождаться загрузки страницы курса

## Ожидаемый результат
- Открывается страница с детальным описанием курса
- URL содержит идентификатор курса (например, `/book?id=1`)
- Все элементы страницы отображаются корректно

## Тип теста
Positive / Functional
