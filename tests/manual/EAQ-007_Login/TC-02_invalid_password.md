# Test Case ID: TC-EAQ-001-02

## Название
Неуспешная авторизация с неверным паролем

## Модуль
Авторизация (Login)

## Приоритет
High

## Предусловия
- Пользователь зарегистрирован в системе
- Пользователь находится на странице авторизации
- Браузер открыт (Chrome)

## Тестовые данные
- Email: test_user@example.com
- Пароль: WrongPassword456

## Шаги
1. Ввести валидный email в поле «Email»
2. Ввести неверный пароль в поле «Пароль»
3. Нажать кнопку «Войти»

## Ожидаемый результат
- Остаёмся на странице авторизации
- Отображается сообщение: «Неверный email или пароль»
- Поле пароля подсвечено красным (или ошибка рядом с формой)
- Пароль не отправлен на сервер в открытом виде
