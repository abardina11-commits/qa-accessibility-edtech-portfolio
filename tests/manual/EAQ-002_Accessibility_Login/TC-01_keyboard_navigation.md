# Test Case ID: TC-EAQ-002-01

## Название
Проверка полной навигации по странице с клавиатуры

## Модуль
Доступность (Accessibility)

## Приоритет
High

## Предусловия
- Открыта главная страница образовательной платформы
- Мышь отключена / не используется
- Браузер: Chrome

## Шаги
1. Нажать клавишу `Tab` несколько раз для перехода между интерактивными элементами
2. Нажать `Shift + Tab` для перемещения назад
3. На всех ссылках нажать `Enter` для перехода
4. На всех кнопках нажать `Enter` или `Space` для активации
5. На чекбоксах и радиокнопках нажать `Space` для изменения состояния
6. На всех выпадающих списках: `Enter` для открытия, стрелки для выбора, `Enter` для подтверждения, `Esc` для закрытия

## Ожидаемый результат
- Все интерактивные элементы (достижимы через `Tab`)
- Видимый индикатор фокуса на каждом элементе
- Порядок перехода логичен: сверху вниз, слева направо
- Нет «ловушек фокуса»: клавиша `Tab` перемещает фокус дальше
- Все элементы активируются с клавиатуры
- При активации ссылки или кнопки происходит ожидаемое действие (переход на страницу, отправка формы)
