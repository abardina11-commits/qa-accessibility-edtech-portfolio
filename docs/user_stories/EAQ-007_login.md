# EAQ-001: User Story — Авторизация пользователя

**Epic:** EPIC 2: Manual Testing

## 📖 User Story
**Как** зарегистрированный пользователь  
**Я хочу** войти в систему, используя email и пароль  
**Чтобы** получить доступ к личному кабинету и курсам

## ✅ Acceptance Criteria
- [ ] При вводе валидных данных → успешный вход
- [ ] При неверном пароле → сообщение «Неверный email или пароль»
- [ ] Пробелы в начале/конце email автоматически обрезаются
- [ ] Кнопка «Войти» блокируется во время отправки запроса
- [ ] Поля имеют связанные `<label>` для screen reader
- [ ] Вход по клавише Enter работает
