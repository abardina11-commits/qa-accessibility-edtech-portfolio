# EAQ-003: User Story — Просмотр каталога курсов

**Epic:** EPIC 2: Manual Testing

## 📖 User Story
**Как** пользователь  
**Я хочу** просматривать и фильтровать каталог курсов  
**Чтобы** быстро найти интересующий меня курс

## ✅ Acceptance Criteria
- [ ] Все курсы отображаются в виде карточек (название, описание, цена, рейтинг)
- [ ] Фильтрация по категориям / цене работает
- [ ] Поиск по названию курса работает
- [ ] Количество найденных курсов корректно обновляется после применения фильтров
- [ ] Пагинация работает корректно (вперёд/назад)
- [ ] Фильтры полностью доступны с клавиатуры (Tab/Enter)
- [ ] При изменении результатов поиска/фильтрации объявляется ARIA live region
