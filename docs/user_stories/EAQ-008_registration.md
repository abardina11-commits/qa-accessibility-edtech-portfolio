# EAQ-002: User Story — Регистрация нового пользователя

**Epic:** EPIC 2: Manual Testing

## 📖 User Story
**Как** новый пользователь  
**Я хочу** зарегистрироваться, указав email и пароль  
**Чтобы** создать аккаунт и получить доступ к платформе

## ✅ Acceptance Criteria
- [ ] При валидном email и пароле (8–20 символов, буквы+цифры) → регистрация успешна
- [ ] При невалидных данных → сообщение об ошибке, объясняющее причину
- [ ] Пароль подтверждён (два поля совпадают)
- [ ] Чекбокс «Согласен с условиями» обязателен
- [ ] Все поля формы имеют явные `<label>` и ARIA-атрибуты
- [ ] После успешной регистрации → редирект на страницу «Подтвердите email»
