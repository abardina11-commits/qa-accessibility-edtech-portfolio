Выбор объекта тестирования (например, демо-сайт https://demoqa.com/)

Браузер: Chrome (последняя версия) с расширениями WAVE, axe DevTools

Инструменты: Lighthouse (встроен), NVDA (опционально), Postman (для API)

Ограничения: слабый ноутбук → использовать лёгкие расширения, не открывать тяжёлые сайты

Данные: тестовые аккаунты, CSV/JSON файлы в examples/test-data/
